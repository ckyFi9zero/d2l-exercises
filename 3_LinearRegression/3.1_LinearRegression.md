### 1. 假设我们有一些数据 $x_1, \dots, x_n \in \mathbb{R}$。我们的目标是找到一个常数 $b$，使得最小化 $\sum_i (x_i - b)^2$ 。

>    a. 找到最优值 $b$ 的解析解。
>    b. 这个问题及其解与正态分布有什么关系?

**a. 解析解推导：**

1. **定义损失函数**：令 $L(b) = \sum_{i=1}^n (x_i - b)^2$。
2. **求导**：对 $b$ 求一阶导数，根据链式法则：
   $$\frac{dL}{db} = \sum_{i=1}^n 2(x_i - b) \cdot (-1) = -2 \sum_{i=1}^n (x_i - b)$$
3. **令导数为零**：寻找极值点：
   $$-2 \left( \sum_{i=1}^n x_i - \sum_{i=1}^n b \right) = 0 \implies \sum_{i=1}^n x_i - nb = 0$$
4. **得出结论**：$b = \frac{1}{n} \sum_{i=1}^n x_i$。
   *即：$b$ 的解析解是这组数的**算术平均值**。*

**b. 与正态分布的关系：**

* **极大似然估计 (MLE)**：假设 $x_i$ 独立同分布于正态分布 $\mathcal{N}(\mu, \sigma^2)$，其概率密度函数为 $p(x) = \frac{1}{\sqrt{2\pi\sigma^2}} e^{-\frac{(x-\mu)^2}{2\sigma^2}}$。
* **似然函数**：$L(\mu) = \prod_{i=1}^n \frac{1}{\sqrt{2\pi\sigma^2}} \exp\left(-\frac{(x_i-\mu)^2}{2\sigma^2}\right)$。
* **对数似然**：$\log L(\mu) = C - \frac{1}{2\sigma^2} \sum_{i=1}^n (x_i - \mu)^2$。
* **结论**：最大化对数似然等价于最小化平方误差和 $\sum (x_i - \mu)^2$。因此，**最小二乘法在概率论上的本质就是假设噪声符合正态分布时的极大似然估计。**

### 2. 推导出使用平方误差的线性回归优化问题的解析解。为了简化问题，可以忽略偏置 $b$（我们可以通过向 $\mathbf{X}$ 添加所有值为1的一列来做到这一点）。

> a. 用矩阵和向量表示法写出优化问题（将所有数据视为单个矩阵，将所有目标值视为单个向量）。
> b. 计算损失对 $w$ 的梯度。
> c. 通过将梯度设为0、求解矩阵方程来找到解析解。
> d. 什么时候可能比使用随机梯度下降更好？这种方法何时会失效？

**a. 矩阵表示：**

* 设特征矩阵 $\mathbf{X} \in \mathbb{R}^{n \times d}$，标签向量 $\mathbf{y} \in \mathbb{R}^n$，权重向量 $\mathbf{w} \in \mathbb{R}^d$。
* 优化目标（损失函数）为：$L(\mathbf{w}) = |\mathbf{y} - \mathbf{Xw}|^2 = (\mathbf{y} - \mathbf{Xw})^\top (\mathbf{y} - \mathbf{Xw})$。

**b. 计算梯度：**

1. **展开损失函数**：
   $L(\mathbf{w}) = \mathbf{y}^\top \mathbf{y} - \mathbf{y}^\top \mathbf{Xw} - (\mathbf{Xw})^\top \mathbf{y} + (\mathbf{Xw})^\top \mathbf{Xw}$
   由于 $\mathbf{y}^\top \mathbf{Xw}$ 是标量，等于其转置 $(\mathbf{Xw})^\top \mathbf{y}$，所以：
   $L(\mathbf{w}) = \mathbf{y}^\top \mathbf{y} - 2\mathbf{w}^\top \mathbf{X}^\top \mathbf{y} + \mathbf{w}^\top \mathbf{X}^\top \mathbf{Xw}$
2. **求梯度** $\nabla_{\mathbf{w}} L(\mathbf{w})$：
   根据矩阵微分公式 $\frac{\partial \mathbf{a}^\top \mathbf{w}}{\partial \mathbf{w}} = \mathbf{a}$ 和 $\frac{\partial \mathbf{w}^\top \mathbf{A} \mathbf{w}}{\partial \mathbf{w}} = 2\mathbf{Aw}$（当 $\mathbf{A}$ 对称时）：
   $$\nabla_{\mathbf{w}} L(\mathbf{w}) = -2\mathbf{X}^\top \mathbf{y} + 2\mathbf{X}^\top \mathbf{Xw}$$

**c. 解析解：**

* 令梯度为 0：$-2\mathbf{X}^\top \mathbf{y} + 2\mathbf{X}^\top \mathbf{Xw} = 0 \implies \mathbf{X}^\top \mathbf{Xw} = \mathbf{X}^\top \mathbf{y}$。
* 如果 $\mathbf{X}^\top \mathbf{X}$ 可逆，则：**$\mathbf{w} = (\mathbf{X}^\top \mathbf{X})^{-1} \mathbf{X}^\top \mathbf{y}$**。

**d. SGD 的讨论：**

* **效果更好时**：当数据集极其庞大（$n$ 很大）时，计算全量梯度的矩阵乘法开销巨大，内存无法负荷。SGD 通过每次只计算一个小批量样本的梯度，可以更快地获得更新方向，节省计算资源。
* **失效时**：

  1. **学习率（LR）不当**：如果学习率过大，模型会在最小值附近震荡而无法收敛；如果过小，收敛极其缓慢。
  2. **数据病态（Ill-conditioned）**：如果特征间存在高度相关性（多重共线性），损失函数的等高线会变得非常扁平，导致 SGD 的路径极其曲折，难以收敛。

### 3. 假设控制附加噪声 $\epsilon$ 的噪声模型是指数分布，也就是说 $p(\epsilon) = \frac{1}{2} \exp(-|\epsilon|)$。

> a. 写出模型 $-\log P(\mathbf{y} \mid \mathbf{X})$ 数据的负对数似然。
> b. 请试着写出解析解。
> c. 提出一种随机梯度下降算法来解决这个问题。哪里可能出错？（提示：当我们不断更新参数时，在驻点附近会发生什么情况）请尝试解决这个问题。

**【步骤详解】**

**a. 负对数似然：**

1. **概率密度**：假设 $y_i = \mathbf{w}^\top \mathbf{x}_i + \epsilon_i$，则 $y_i$ 的概率密度为 $p(y_i \mid \mathbf{x}_i, \mathbf{w}) = \frac{1}{2} \exp(-|y_i - \mathbf{w}^\top \mathbf{x}_i|)$。
2. **似然函数**：$P(\mathbf{y} \mid \mathbf{X}) = \prod_{i=1}^n \frac{1}{2} \exp(-|y_i - \mathbf{w}^\top \mathbf{x}_i|)$。
3. **负对数似然**：
   $-\log P(\mathbf{y} \mid \mathbf{X}) = -\sum_{i=1}^n \left( \log \frac{1}{2} - |y_i - \mathbf{w}^\top \mathbf{x}*i| \right)$
   忽略常数项后，优化目标为最小化：**$L(\mathbf{w}) = \sum*{i=1}^n |y_i - \mathbf{w}^\top \mathbf{x}_i|$** （即 $L1$ 损失或平均绝对误差 MAE）。

**b. 解析解：**

* **结论**：该问题**没有像线性回归那样的封闭矩阵形式解析解**。
* **原因**：因为绝对值函数的导数在零点是不连续的，无法直接通过线性代数方程组（Normal Equations）求解。这通常需要使用线性规划（Linear Programming）或迭代优化算法。

**c. SGD 算法与“尖点”处理：**

* **SGD 策略**：每次迭代随机抽取一个样本 $(x_k, y_k)$，更新公式为：
  $\mathbf{w} \leftarrow \mathbf{w} - \eta \cdot \text{sign}(\mathbf{w}^\top \mathbf{x}_k - y_k) \cdot \mathbf{x}_k$。
* **处理“尖点”（不可微点）的方法**：

  1. **次梯度法（Subgradient Method）**：在不可微点定义一个伪梯度（通常在 0 处取 0）。
  2. **平滑处理（Smoothing）**：使用 **Huber Loss** 代替单纯的 $L1$ 损失。Huber Loss 在误差较小时使用平方误差（平滑可微），在误差较大时使用绝对误差（稳健）。
  3. **动态学习率（Adaptive Step Size）**：随着迭代次数增加不断减小学习率 $\eta$，防止在尖点附近来回跳变，强制模型在极值点附近停下来。
